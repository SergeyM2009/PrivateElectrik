# CSS Modules Emmet

Расширение для VS Code, которое заменяет Emmet-разметку на JSX с CSS Modules.

## Использование

Напиши:


.example


Нажми:


Shift + Enter


Получишь:

```jsx
<div className={styles.example}></div>

Также работает:

section.hero

Результат:

<section className={styles.hero}></section>
Поддержка

Работает с React / Next.js и CSS Modules.
